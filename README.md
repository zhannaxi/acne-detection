# Acne Detection System with Grad-CAM

## Overview
A deep learning model that detects acne severity levels from facial images using ResNet-50 and visualizes predictions with Grad-CAM.

## Results
- Test Accuracy: 75%
- Model: ResNet-50 (Transfer Learning)
- Explainability: Grad-CAM visualization

## Dataset
ACNE04 - 1457 facial images, 4 severity levels

## Installation
pip install torch torchvision torchcam gradio scikit-learn seaborn

## Usage
Run all cells in acne_detection.ipynb
